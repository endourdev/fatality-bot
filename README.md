# Getting Started with DiscordJs


## A simple bot by Fofhgit
### Build with discord.js

````
Create a config.json file and type this :
````
````
{
    "token": "your-token",
    "clientId": "your-client-id"
}
````

### Start your bot with
````
npm install
npm run start
````

### Library needed
discord.js
````
npm install discord.js
````
